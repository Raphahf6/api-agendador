# API Agendador

API backend para o sistema de agendamento, criada para apoiar fluxos de agenda, clientes, servicos e integracoes do produto Agendador/Horalis.

## Papel no ecossistema

Este repositorio complementa o frontend `agendador`, separando regras de backend e integracoes que nao devem ficar no navegador.

## Contexto de produto

- Apoio a agendamentos e operacao de profissionais.
- Base para integracoes externas e regras server-side.
- Parte de uma linha de SaaS vertical para negocios locais.

## Stack principal

- Python
- API backend
- Integracoes de agenda e automacao

## Observacao

A documentacao tecnica pode evoluir junto com o backend. Como boa pratica, mantenha aqui exemplos de variaveis de ambiente, rotas principais e instrucoes de deploy quando o projeto estiver em producao.
